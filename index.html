How to Play
Start with five numbers.

Combine them using addition, subtraction, multiplication, and division to make the Goal Number.

You must use all five numbers. There are usually many different solutions!

If you get stuck, click Reset to start the puzzle over. If you feel like you can't solve the puzzle, click Give Up to end the game and view the solution.
